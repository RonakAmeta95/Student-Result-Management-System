# STUDENT RESULT MANAGEMENT SYSTEM

A Python-based console application for managing student records and results. This project allows users to add, search, update, and delete student records while automatically calculating total marks, grades, and pass/fail status.

## FEATURES

- Add Student Records
- Search Student Results
- Update Student Records
- Delete Student Records
- Calculate Total Marks
- Generate Grades
- Display Pass/Fail Status
- Input Validation for Marks (0–100)
